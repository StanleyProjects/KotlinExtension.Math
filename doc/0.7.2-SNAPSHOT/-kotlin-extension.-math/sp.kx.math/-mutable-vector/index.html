MutableVector

class MutableVector(val start: MutablePoint, val finish: MutablePoint) : Vector(source)

A mutable implementation of the Vector type.

Usage:

val vector = MutableVector(
    start = MutablePoint(x = 1.2, y = 3.4),
    finish = MutablePoint(x = 5.6, y = 7.8),
)
vector.start.x = 2.0
vector.finish.y = 3.0
