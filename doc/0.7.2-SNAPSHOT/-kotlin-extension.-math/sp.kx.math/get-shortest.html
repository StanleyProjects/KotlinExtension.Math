fun getShortest(xStart: Double, yStart: Double, xFinish: Double, yFinish: Double, xTarget: Double, yTarget: Double): Double(source)

The function calculates the shortest distance from point to segment. It is up to the segment, and not the length of the perpendicular to the straight line!

Usage:

val a = pointOf(x = 2, y = 3)
val b = pointOf(x = 1, y = 1)
val c = pointOf(x = 3, y = 1)
val value = getShortest(
    xStart = b.x,
    yStart = b.y,
    xFinish = c.x,
    yFinish = c.y,
    xTarget = a.x,
    yTarget = a.y,
)
assertEquals(2.0, value)
