fun getShortestPoint(xStart: Double, yStart: Double, xFinish: Double, yFinish: Double, xTarget: Double, yTarget: Double): Point(source)

The function calculates the shortest Point from point to segment. It is up to the segment, and not the Point of the perpendicular to the straight line!

Usage:

val a = pointOf(x = 2, y = 3)
val b = pointOf(x = 1, y = 1)
val c = pointOf(x = 3, y = 1)
val s = getShortestPoint(
    xStart = b.x,
    yStart = b.y,
    xFinish = c.x,
    yFinish = c.y,
    xTarget = a.x,
    yTarget = a.y,
)
assertEquals(2.0, s.x)
assertEquals(1.0, s.y)
