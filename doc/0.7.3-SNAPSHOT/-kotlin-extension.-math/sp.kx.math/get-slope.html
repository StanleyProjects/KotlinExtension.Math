fun getSlope(aX: Double, aY: Double, bX: Double, bY: Double): Double(source)

A line's steepness is measured by the absolute value of its slope. The larger the value is, the steeper the line. Given a slope, it is possible to determine the direction of the line that a slope describes based on its sign and value.

Usage:

val a = pointOf(x = 1, y = 1)
val b = pointOf(x = 3, y = 2)
val value = getSlope(
    aX = a.x,
    aY = a.y,
    bX = b.x,
    bY = b.y,
)
assertEquals(0.5, value)
